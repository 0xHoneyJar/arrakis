### Description
Built for protocols with security and compliance requirements. Row-level security, audit trails, and architecture designed for scale.

### Security Features

#### Row-Level Security (RLS)
Every database query is scoped to your community. Complete tenant isolation at the database level â€” not application-level filtering.

#### Audit Trail
Full logging of all administrative actions. Who changed what, when. Export logs for compliance review.

#### Two-Tier Architecture
Our two-tier chain provider ensures token-gating never goes down:

| Tier | Purpose | Availability |
|------|---------|--------------|
| **Tier 1: Native Reader** | Basic balance/ownership checks | Always available |
